Mumble Info & Setup


Mumble is a free voice chat application that uses military-grade encryption.

Even with encryption, Mumble has the lowest latency out of any VoIP app.
This makes it the voip of choice for professional competitive gamers.

Connecting

After you have downloaded Mumble, follow the install wizard to completion.

To connect to the server, click on the globe icon to the top left.
Click on Add New and new window will appear.
You will be asked for our server info below.

  • Label: JustSomeGamers
  • Address: mumble.justsomegamers.com
  • Port: 6052
  • Username: Use a screen name

Alternativly, you can use this button to connect.

Admin Contact: justsomegamerscommunity@gmail.com
